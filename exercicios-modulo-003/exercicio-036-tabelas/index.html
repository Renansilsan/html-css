Como criar uma tabela

Iniciamos incluindo a TAG TABLE, para indicar que esta sendo criando uma tabela, dentro incluímos a TAG TR, que indica linhas da tabela, incluimos dentro da TR a TAG TD que são as TAGs onde incluímos os dados da tabela, para configurar a tabela e deixa-la com as cédulas juntas, usar a TAG BORDER-COLLAPSE: COLLAPSE.
