Onde ultilizar o estilo externo

Este estilo de CSS é recomendada para modificação em massa e em diversas páginas, para ultiliar o estilo externo teremos que criar um arquivo CSS e esse arquivo iremos incluir no HTML. Primeiramente criar um arquivo CSS e incluir todas modificações, no HTML, na área HEAD, incluir a TAG LINK.CSS, dentro da TAG incluir o arquivo CSS e assim será em todas as páginas que queira a configuração feita no arquivo CSS.
