Onde ultilizar estilo inline

O estilo inline não possibilita alterações em massa, por isso é recomendado ultiliza-lo para fazer alterações pontuais. Incluir a TAG STYLE para iniciar as alterações com CSS,para alterar a cor do fundo do site incluir a TAG BACKGROUND-COLOR e posteriormente escolher a cor, para alterar cor da fonte ultilizar a TAG COLOR e informar a cor desejada, para justificar o texto, deixar alinhado o espaçamento ultilizar a TAG TEXT-ALIGN e depois TAG JUSTIFY ou escolher a formar que preferir organizar a escrita.
