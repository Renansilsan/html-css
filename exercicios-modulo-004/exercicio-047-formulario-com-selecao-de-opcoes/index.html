Formulário com lista de opções


Criando formulários com Select, Datalist e Textárea

  • Select - Podemos usar o SELECT para construir formulários com opções que não podem ser alteradas, neste exemplo temos uma lista feita com estados, como só existem uma certa lista de estados, não há como informar outra opção. Para construir o formulário começamos incluindo um LABEL e abaixo a TAG SELECT lincando os IDs, na linha abaixo vamos incluir um OPTGROUP, para agrupar opções, neste exemplo separamos as opções de acordo com cada região, dentro de OPTGROUP começamos a incluir os dados com a TAG OPTION e informamos a sígla de cada estado dentro de VALUES;

  • Datalist - Diferentemente do select, usamos o DATALIST para formulários em que pode ser incluído informações pelo usuário, temos o exemplo em que o usuário deve preencher a profissão, como há um número muito grande de profissões, colocamos algumas opções, porém o usuário pode incluir alguma caso a que ele deseja selecionar não esteja dentre as opções. Para construir a tabel iniciamos com o LABEL, na linha seguinte adicionameos o INPUT:TEXT e lincamos os IDs, na linha subsequênte colocamos a TAG DATALIST e lincamos os IDs, agora começamos a incluir os dados e cada dado será incluido dentro de DATALIST e dentro da TAG OPTION;

  • TEXTAREA - Para adicionar uma caixa de texto em que o usuário possa deixar uma mensagem suamos a TAG TEXTAREA. Para imputar a caíxa de texto vamos primeiro incluir um LABEL e na linha seguinte a TAG TEXTAREA, depois lincamos os IDs. É aconselhavel que o METHODE da lista seja POST para que não haja problemas com envio de dados caso o texto escrito na caíxa seja muito grande.
