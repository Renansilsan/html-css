Como ultiliza Media Query para diferentes resoluções

Para criar um layout adaptado para cada resolução, celular, tablets ou computador, criamos este exemplo usando Media Query

Dentro de Style criamos uma regra sendo @MEDIA SCREE AND (ORIENTATION: PORTRAIT) para ajustar a resolução para celulares e tablets, e incluimos o @MEDIA
SCREEN AND (ORIENTATION: LANDSCAPE) para configurar o layout da tela que será exibida nas telas de computadores e televisões.
