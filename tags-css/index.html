TAGs CSS

  • STYLE - TAG base para dar inicio as configurações.
  • BACKGROUND-COLOR - Ultilizada para alterar a cor de funda da página.
  • COLOR - Ultilizada para alterar cor da fonte.
  • TEXT-ALIGN - Ultilizada para formatação do alinhamento do texto.
  • FONT-SIZE - Ultilizada para alterar o tamanho da fonte.
  • FONT-FAMILY - Ultilizada para alterar o estilo da fonte.
  • BACKGROUND-IMAGE - Para adicionar um fundo em degradê.
  • LINEAR-GRADIENT - Para indicar o formato do degrade.
  • BACKGROUND-ATTACHMENT - Para indicar que o degrade ficará fixo em toda a página.
  • MAIN - Caixa de texto onde será descorrido o assunto principal do site.
  • BORDER-RADIUS - Para mudar as bordas da caixa de texto, exemplo 10px.
  • BOX-SHADOW - Para inserir uma sobra na caixa de texto, tendo que inserir as medidas do sobreamento e a cor ultilizada, exemplo 0px 0px 7px 5.Black.
  • WIDTH - Largura da caixa de texto,
  • Height - Altura da caixa de texto.
  • Border - Bordas que ficam ao redor da caixa de texto.
  • PADDING - Expaço dentro da caixa de texto entre o texto e a borda.
  • MARGIN - Expaço ao redor da caixa de texto, fora da caixa, para centralizar a caixa, selecionar AUTO.
  • Outline - Linha ao redor da do texto, em cima da borda, contorno.
  • TEXT-SHADOW - Para inserir uma sobra no texto, tendo que inserir as medidas do sobreamento e a cor ultilizada.
  • FONT-WEIGHT - Para definir o peso da fonte, largura, podendo usar LIGHTER (fonte mais magra), NORMAL (Fonte comum), BOLD (Fonte pesada), BOLDER (Fonte super pesada) e também temos a opção de descrever poor via númerica de 100-900, sendo que nem todas as fontes tem todas essas configurações.
  • FONT-STYLE - Para definir configurações como Itálico.
  • TEXT-DECORATION - Para configurações como UNDELINE.
  • TEXT-INDENT - Para calcular o espaçamento do parágrafo.
  • @FONT-FACE-FONT-FAMILY - Seletor para incluir fontes externas.
  • FORMAT - Para incluir o formato do arquivo da fonte externa inclusa.
  • # - ID
  • . - Class
  • : - Pseudoclass
  • :: - Pseudoelement
  • > - Pertence ao
  • :HOVER - Pseudoclasse para configurar o que acontecerá após o mouse passar sobre, será sempre acompanhado de : para indicar que será uma pseudoclasse.
  • :ACTIVE - Psedoclasse para para links, para configurar a cor que o link ficará após ser clicado.
  • ::BEFORE - Para incluir algo antes de um elemento.
  • ::AFTER - Para incluir algo depois de um elemento.
  • HEADER - Seção para inclusão de cabeçalho em uma página.
  • NAV - Seção para inclusão de links em uma página.
  • MAIN - Seção para inclusão do conteudo principal de uma página.
  • ARTICLE - Seção para agrupar conteudo que são relacionados entre si.
  • FOOTER - Seção de rodapé, conteudo final de um site.
  • BACKGROUND-POSITION - Ultilizado para posicionar uma imagem, podendo selecionar combinações entre Right, Left, Center, Bottom e Top.
  • BACKGROUND-SIZE - Para definir o tamanho da imagem do fundo.
  • BACKGROUND-REPEAT - para as repetições da imagem, tendo Y, X e NO-REPEAT como algumas da variações que podem ser usadas.
