Como deixar uma tabela responsiva

Quando uma tiver um número muito grande de linhas ou colunas que faça com que o site tenha fazer uma rolagem lateral, podemos resolver da seguinte maneira. 1 - Evelopar toda a tabela em uma DIV 2 - Criar um ID/CLASS para a DIV e 3 - Configurar com a TAG OVERFLOW: AUTO, caso o número de linhas seja muito grande usar OVERFLOW-Y caso seja o número de colunas usar OVERFLOW-X, fazendo isso será criado uma barra de rolagem apenas para a tabela.
