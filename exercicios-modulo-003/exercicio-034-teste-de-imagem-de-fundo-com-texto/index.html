O que fizemos

Neste caso ultilizamos a TAG HEIGHT: 100VH para que a imagem ocupe toda a tela, ultilizamos o BACKGROUND-POSITION: CENTER CENTER para centralizar a imagem, usamos o BACKGROUND-REPEAT: NO-REPEAT, ultilizamos o BACKGROUND-SIZE: COVER para que a imagem cubra toda a tela, mesmo que a imagem seja cortada ainda ocupará toda a tela e o BACKGROUND-ATTACHMENT: FIXED para que a imagem fique fixa na tela, o texto será rolado porém a imagem de fundo estará fixa.
