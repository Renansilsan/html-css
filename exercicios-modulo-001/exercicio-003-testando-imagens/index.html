Testando imagem

Abaixo você verá uma imagem que esta na mesma página.

Para incluir uma imagem no site, ultlizar a TAG IMG, incluir a imagem entre os parênteses de SRC (ultilizando Control + Espaço), entre os parênteses de ALT descrever o nome da imagem.

Para incluir uma imagem externa, devemos copiar o link da imagem e incluir no código.

Formatos de imagem

  • JPG - Para grandes imagens por ter alta compactação, arquivos mais leves.

    • Largura média de uma imagem para um site é em torno de 1500 e a resolução média ao redor de 75, para arquivos JPG

  • PNG - Para imagens menores ou que não tenha fundo, permite transparência

    • Largura média de uma imagem para um site é em torno de 400 e a resolução nunca pode ser maior que a resolução da imagem original, para arquivos PNG

Plataforma para edição de imagens

Ultilizar o GIMP.
