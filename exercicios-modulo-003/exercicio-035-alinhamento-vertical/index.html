Alinhamento de verticais

O que fizemos

Fizemos este exemplo para demonstrar algumas TAGs de posicionamento de imagens. Iniciamos com a TAG POSITION que pode ser selecionado entre RELATIVE, que terá como referência o canto superior esuqeurdo da tela e a partir de sua posição de partida iniciar as modificações desejadas, ja a opção ABSOLUTE terá 2 comportamentos, quando estiver dentro de uma caíxa com posicionamento STATIC ou sozinha na tela, ela sempre seguirá seu alinhamento inicial no canto superior esquerdo da tela. Caso esteja dentro de uma caíxa que não seja STATIC, ele sempre acompanhará a formatação da caixá que está a sua volta. Com o ABSOLUTE podemos configurar o elemento com as TAGs LEFT e TOP, podendo ultilizar % para definir o posicionamento, usamos também a TAG TRANSFORM: TRANSLATE para alinhar a imagem perfeitamente com o alvo.
