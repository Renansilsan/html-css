Como incluir uma imagem como fundo

Neste caso temos 3 exemplos de fundo, 1- Ultilizando BACKGROUND-COLOR, 2 - Ultilizando BACKGROUND-IMAGE, LINEAR-GRADIENT e selecionando a direção e as cores do degradê e o 3 - Ultilizando uma imagem, para incluir uma imagem como fundo temos que incluir a TAG BACKGROUND-IMAGE + URL, entre PARÊNTESES e posteriomente entre '' incluir o caminho para chegar até o arquivo com a imagem desejada.
