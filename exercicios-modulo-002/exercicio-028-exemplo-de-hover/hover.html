Como incluir um texto escondido

Primeiro vamos configurar a página que aparecerá antes de passar o mouse, começameos incluindo o seletor DIV + > "para indicar que pertende ao DIV" + a TAG DISPLAY "para fazermos configurações como NONE ou BLOCK para que o texto fique visivel ou não" neste caso incluiremos o NONE, para informar que este texto não ficará visivel.

Segundo passo, criar um seletor DIV + : + HOVER, fazer as configurações para que quando o mouse passar sobre as DIVs elas ficarão da forma que foi configurada, neste caso elas ficarão com a cor vermelha de fundo e a fonte ficará branca.

Por ultimo incluir DIV + : + HOVER + > + P para indicar que quando o mouse passar dentro da DIV incluida o texto configurado neste seletor aparecerá, incluir a TAG DISPLAY e selecionar BLOCK para indicar que o texto tem que ser desbloqueado quando o mouse passar.
