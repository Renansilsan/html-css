Links externos

Você pode acessar meu repositório público no Github

Para incluir um Hyperlink no código, selecionar o texto, envelopar (Conrol + Shift + P), incluir a TAG A (Ancora) e adicionar o link dentro dos parenteses após a palavra HREF.

Para fazer com que o link externo seja aberto em uma nova aba, incluir a TAG TARGET, selecionar _BLANK e depois a TAG REL e no parenteses EXTERNAL, para indicar que esse link externo não pertence ao site inicial.

Links internos

Essa é a primeira página do site, se preferir pode acessar também a minha segunda página

Para indicar ao navegador a próxima página que será aberta, continuação desta página, usar a TAG A (ãncora) e incluir a página ultilizando Control + Espaço, os arquivos tem que estarem na mesma pasta.

Ultilizar a TAG REL entre parenteses NEXT, para indicar ao navegador que a página incluida na âncora será uma pagina sequencial a essa.

Ultilizar a TAG TARGET e selecionar SELF, para informar o navegador que a navegação será interna.
