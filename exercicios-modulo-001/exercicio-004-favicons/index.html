Olha o favicon em cima

Para adicionar o favicon, icone de imagem do link;

  1. Na área Head, incluir a TAG LINK.FAVICON ;

  2. Apagar o texto que está entre aspas;

  3. Clicar CONTROL + ESPAÇO e adicionar o arquivo ICO( somente arquivos no formato ICO conseguirão ser reconhecidos).


Sites para busca de Favicons;

Icon.archive - Site para pesquisa de icones.

Favicon.cc - Site para criação de icones.

Favicon.10 - Site para criação e conversão de arquivos de outros formatos para o formato ICO.
