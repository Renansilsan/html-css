Imagens dinâmicas

Tente abrir essa imagem em diversos dispositivos diferentes ou simplesmente aumente e diminua o tamanho do navegador. Para que a imagem seja carregada de acordo com o dispositivo ultilizado, celular, tablet ou computador, por exemplo, ultilizar a TAG SOURCE:MEDIA:TYPE, trocar o escrito em media para "(max-width: informar a dimenção maxima para que a imagem seja trocada)", em SRCSET descrever o nome do arquivo (não aceita o comando Control + Espaço, terá que descrver o atalho para da pasta até o arquivo) e TYPE informar tipo do arquivo "Imagem/PNG". por exemplo. Essa TAG é ultilizada para que o sistema modifique o tamanho do arquivo de acordo com o dispositivo de navegação, caso seja um celular usará imagem pequena, se for um Tablet será uma imagem média ou caso seja um computador será a imagem grande. Sempre ordenar as imagens da maior para a menor, apenas dessa forma o código funcionará,

Reproduzindo áudio em HTML

Vamos aprender a reproduzir áudio em HTML

Para incluir audio no site, iniciamos unsando a TAG ÁUDIO apagar a TAG SOURCE incluída automaticamente, dentro da TAG ÁUDIO podemos incluir a TAG PRELOAD, dentro de Preload teremos 3 opções, AUTO "o site só aparecerá completo após todo o audio ser carregado", METADATA "o site carregará apenas algumas informações, como data e nome do arquivo" e NONE "não carregrá qualquer informação apenas quando o usuario clicar em play" TAG AUTOPLAY "para que o audio seja reproduzido automaticamente, porém alguns navegadores não são compativeis", caso a TAG AUTOPLAY não funcione, incluir a TAG CONTROLS para que apareça um menu onde o usuario do site possa dar play no audio e também podemos ultilizar a TAG LOOP, para que o áudio seja reproduzido novaente quando finalizar. Dar enter e na próxima linha incluir a TAG SOURCE:SRC, incluir o arquivo usando Control + Espaço e na TAG TYPE incluir ÁUDIO + /e o formato do arquivo, por exemplo Áudio/mpeg.

Incluir mais de um formato de áudio no código, pois se o nagegador do usario não conseguir reproduzir o primeiro formato, o navegador tentará reproduzir o segundo formato.

Tipos de formato de áudio

Os mais comuns são MP3/MPEG, OGG e WAV, por serem mais leves e de fácil compactação arquivos com formato MPEG e OGG são mais indicados para sites enquandto o formato WAV tem qualidade melhor porém são mais pesados e inviaveis para incluir em sites.

Direitos autorais em áudio

Uma plataforma para pesquisa de áudios para incluir em um site sem direitos autorais é o Youtube Studio, entrar no youtube, menu principal, youtube studio, blilioteca de áudio e filtrar. Áudios que são NECESSÁRIO ATRIBUIÇÃO precisam ter um termo, incluso no proprio site, junto com o código por questões de liberação de direitos autorais, áudios ATRIBUIÇÃO NÃO NECESSÁRIA está leberado para inclusão sem necessidade de inclusão do termo.
