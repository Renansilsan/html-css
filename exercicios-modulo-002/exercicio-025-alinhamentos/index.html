Como alinhar

Para fazermos o alinhamento das linhas usaremos a TAG TEXT-ALIGN e selecionar a direção RIGHT, LEFT, CENTER e JUSTIFY são as mais ultilizadas. Também podemos usar a TAG TEXT-INDENT para configurar o espaçamento do parágrafo, 30px é uma medida recomendada.
