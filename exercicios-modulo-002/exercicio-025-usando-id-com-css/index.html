Como criar um seletor ID

Um seletor ID é ultilizado para criar uma classficação especifica em um elemento especifico, dando a possibilidade de alterar somente o texto que esteja marcado com o ID selecionado sobre ele.

Para criar o seletor devemos incluir, detro do elemento que deseja a configuração composta nesse seletor, a TAG ID e depois criar um nome para o ID, para configurar dentro do CSS, incluir o ELEMENTO+#+NOME DO ID.

Como criar um seletor CLASS

Um seletor CLASS é ultilizado para criar uma classficação especifica para um grupo de elemento, dando a possibilidade de alterar somente o texto que esteja marcado com o CLASS selecionado sobre ele.

Para criar o seletor devemos incluir, detro do elemento que deseja a configuração composta nesse seletor, a TAG CLASS e depois criar um nome para o CLASS, para configurar dentro do CSS, incluir o .+NOME DO CLASS.
