Como organnizar uma tabela

Temos este exemplo para praticar a confecçaõ de uma tabela grande. Introduzimos a TAG TABLE para dar inicio a tabela, uma tabela será divididas em 4 partes, 1 - CAPTION (título da tabela), 2 - THEAD (cabeçalho da tabela), 3 - TBODY (informações do corpo, os dados da tabela) e 4 - TFOOT (rodapé da tabela). Para incluir os dados na tabela usaremos as TAGs TH e TD, sendo TH para os títulos e TD para os dados.

Tabela zebrada

Para criarmos uma tabela zebrada, usaremos a PSEUDOCLASSE :NTH-CHILD e entre as aspas informar o numero que deseja, por exemplo, 2n fará com que intercale entre uma linha pintada e outra não pintada, se selecionar 3n, uma linha será pintada, 2 ficarão em branco a próxima será pintada e assim por diante. Podemos também colocar um número negativo, para que o preenchimento de cores se inverta e também podemos selecionar por impar (ODD) e PAR (even).

Fixando o cabeçalho

Para fixarmos o cabeçalho precisamos usar a TAG POSITION:RELATIVE em TABLE, depois vamos fazer as configurações do TH que são os dados do cabeçãlho, iniciamos com a TAG POSITION:STICKY depois incluimos a TAG TOP: -1px, para que o cabeçalho comece a descer quando chegar em -1px da tela e alteramos o BACKGROUNDCOLOR para a mesma cor contida no fundo do cabeçalho.
