Onde ultilizar o estilo interno/local

Esse estilo de técnica é bastante ultilizado para fazer mudanças em massa dentro de um página, sua vantagem é conseguir fazer diversas alteraçõe sem precisar faze-las individualmente, porém a desvantagem é apenas conseguir fazer na mesma página e não conseguir alterar outras páginas contidas no mesmo site

Para ultilizar essa técnica devemos incluir a TAG STYLE na parte HEAD do código e depois a TAG BODY incluindo colchetes ({) posteriormente, entre os colchetes colocar as TAGs e as alterações que deseja fazer, caso deseje alterar os H1s de uma página, fazer o mesmo processo apenas alterando o BODY pelo H1 e a mesma lógica permance para H2, paragrafos e etc.
