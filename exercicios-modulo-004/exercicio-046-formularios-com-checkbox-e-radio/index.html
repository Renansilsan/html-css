Como construir um formulário com caíxas de marcação

  • INPUT:RADIO - Para construir caíxas onde apenas uma alternativa pode ser marcada, sempre colocar o mesmo nome dentro de NAME e incluir um VALUE, com qualquer digíto desde que não seja igual de outro input.;
  • INPUT:CHECKBOX - Para construir caíxas onde mais de uma opção pode selecionada, a cada item incluído incluir um LABEL para cada INPUT;
  • Sempre incluir os inputs dentro de um FORM e dentro de form criar um FIELDSET, para agrupar caíxas que pertecem ao mesmo assunto, podemos també criar um título para o agrupamento usando a TAG LEGEND e colocando o nome que desejar.

Formulário para anexar arquivos

Como elaborar?

  • INPUT:FILE será a TAG ultilizada para construção de formulários para que o visitante possa anexar arquivos. OBS quando incluir este tipo de formulário temos que deixar sempre o METHODE="POST" para que o site consiga carregar todo o arquivo.
