O que são Media Quary?

Media Quaries são estilos de personalização especificos para certas formas de navegação, neste caso temos a configuração da tela de navegação do site e uma configuração para impressão, sem menus, links e com uma fonte mais adaptada para leitura em um papel.

Criando Media Media Quary

Para ultlizar esta técnica devemos criar tipos de arquivos CSS diferentes e cada um com a configuração que se adapte aos tipos de navegação desejado. Iniciamos incluindo o LINK:CSS, incluir depois a TAG MÉDIAS e para onde será as configurações contidas neste arquivo, neste caso SCREEN para a tela principal e PRINT para a tela de impressão. OBS caso queira adiciona algo em uma tela que não esteja no HTML, incluir o PSEUDOELEMENTO ::AFTER, dentro dele incluir CONTENT + texto.
